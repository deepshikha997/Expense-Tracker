# Expense-Tracker
Full Stack Expense Tracker using MERN Stack
## Features
- Add expenses
- View all expenses
- Delete expenses
- Automatic total calculation

## Tech Stack
- Frontend: React (Vite)
- Backend: Node.js, Express.js
- Database: MongoDB
- API Communication: Axios

## How to Run Locally

### Backend
```bash
cd backend
npm install
npm start
